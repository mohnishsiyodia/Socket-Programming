
README

1. The zip file consists the solution file with 1 server and 3 clients connecting over TCP/IP protocol using socket programming.
2. The path for solution file KIFS_Test_Mohnish_Siyodia >> Socket_Server_Client >> Socket_Server_Client.sln
3. The directory Application Executables consists of the release binaries of server and clients.
4. Kindly start the server first, if not then the clients cannot connect to a server and client will shut down.

Scenarios covered:

1. Server - Client connection using socket programming
2. Server print - Client connecting to the Server
3. Server print - Client disconnecting from the Server
4. Server sending heartbeat every 5 minutes to all the connected clients and counter of the heartbeat increments by 1 every 1 minute
